# Notes App

A simple and lightweight notes application built with React.

## Features

- Create, read, update, and delete notes
- Simple and intuitive user interface
- Local storage support for persisting notes
- Fast and responsive performance

## Installation

1. Clone the repository
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```bash
    npm start
    ```

## Usage

- Click "New Note" to create a note
- Edit note content by clicking on the note
- Delete notes using the delete button
- Notes are automatically saved

## Technologies Used

- React
- HTML/CSS
- JavaScript
